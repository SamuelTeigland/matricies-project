# Matricies Solver

## Purpose

The purpose is to be able to solve complex matricies for imaginary numbers.

## Installation

### 1. Create a virtual environment

We need to be able to create a virtual environment in Python in order to host the website.  Virtual environments allow us to do that.

```powershell
py -m venv .venv
```

### 2. Activate the virtual environment

Next, we need to activate the virtual environment with this command.

```powershell
.venv\Scripts\activate
```

### 3. Install Flask

Finally, we need to install Flask.  Flask is a light-weight Python web framework.  This will give us the structure for our matricies website.

```powershell
pip install Flask
```
